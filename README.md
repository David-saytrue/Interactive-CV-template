# Interactive CV Template

This is an interactive CV (Curriculum Vitae) template built with HTML, CSS, and JavaScript.

## Features

- Responsive design for various devices
- Interactive elements for a dynamic user experience
- Customizable sections for personal information, skills, experience, etc.

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Customize the content in `index.html` and `style.css` to fit your needs.

## Project Structure

- `index.html`: The main HTML file containing the CV structure.
- `style.css`: CSS file for styling the CV.
- `assets/img/`: Folder containing images used in the CV.
- `assets/js/script.js`: JavaScript file for interactive functionality.
